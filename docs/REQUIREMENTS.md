# 미션 - 점심 뭐 먹지

## 프로젝트 설명

- 캠퍼스 주변의 점심 식사 스팟 목록을 관리하는 앱을 만든다.

---

## 기능 목록

### 1. 기본적인 기능

- [x] 음식점 목록을 확인

  - [x] 카테고리별 필터링 결과 확인

  - [x] 이름 거리순 음식점 목록 정렬

- [x] 버튼 클릭 시 모달창 출력

- [x] 음식점 목록에 아래 항목들을 입력해서 새로운 음식점을 추가

  - [x] 카테고리 (**필수**)

  아래와 같은 항목들을 선택해서 추가할 수 있다.

  - [x] 한식

  - [x] 중식

  - [x] 일식

  - [x] 아시안

  - [x] 양식

  - [x] 이름 (**필수**)

  - [x] 거리(도보 이동 시간) (**필수**)

  아래와 같은 항목들을 선택해서 추가할 수 있다.

  - [x] 5분

  - [x] 10분

  - [x] 15분

  - [x] 20분

  - [x] 30분

  - [x] 설명 (**옵션**)

  - [x] 참고 링크 (**옵션**)

- [x] 로컬스토리지를 통해 음식점 정보 데이터 유지
